# loski-share

### gtloski的分享项目，目的是创建一个收集分享自己喜欢的东西的网站，包括音乐，电影，图片，技术等东西。

### Maven 启动 `clean tomcat7:run`
	
### 访问地址：http://localhost:8080/share/main/login.html
